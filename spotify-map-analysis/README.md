# Global Music Mapping — Spotify Data Analysis

Data processing project focused on transforming and enriching music datasets by linking songs with geographic information through country codes.

## Objective

Create a structured dataset connecting songs, artists, and countries by merging multiple data sources and preparing information for geographic analysis.

## Process

1. Import playlist dataset.
2. Load country metadata table.
3. Merge datasets using ISO country codes.
4. Clean missing values and inconsistencies.
5. Generate enriched dataset ready for mapping and analysis.

## Techniques Used

- Data merging (joins)
- Data cleaning
- Feature enrichment
- Dataset transformation

## Technologies

- Python
- Pandas
- Excel datasets
- Data preprocessing

## Output

Dataset including:

- Song name
- Artist
- Genre
- Duration
- Country code
- Country name
