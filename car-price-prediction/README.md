# Used Car Price Prediction — Machine Learning Model Comparison

Machine learning project focused on estimating vehicle market prices using historical data while balancing prediction quality, training time, and inference speed.

## Objective

Develop and compare predictive models capable of estimating used car prices efficiently for deployment in a customer-facing application.

## Dataset

Historical used-car dataset including:

- Vehicle type
- Registration year
- Gearbox type
- Power
- Mileage
- Model specifications
- Historical prices

## Methodology

### Data Preparation
- Removal of missing and invalid price values
- Conservative outlier filtering
- Date feature engineering
- Handling categorical variables
- Dataset cleaning and transformation

### Feature Engineering
- Extraction of temporal features from dates
- Encoding categorical variables
- Construction of modeling pipeline

### Models Evaluated
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

### Evaluation Criteria
- Prediction accuracy (RMSE)
- Training time
- Prediction speed

## Results

Model comparison highlights trade-offs between accuracy and computational efficiency, supporting model selection under real-world constraints.
